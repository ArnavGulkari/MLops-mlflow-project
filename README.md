## Project Overview

This project demonstrates a complete **Machine Learning Operations (MLOps) pipeline** that manages the entire lifecycle of a machine learning model, including data versioning, experiment tracking, model training, and deployment.

The pipeline integrates modern open-source tools to ensure reproducibility, scalability, and efficient model management.

---

## Technologies Used

* Python
* MLflow
* DVC (Data Version Control)
* BentoML
* Git & GitHub
* Scikit-Learn
* Pandas
* NumPy

---

## Features

* Data versioning using **DVC**
* Experiment tracking using **MLflow**
* Model packaging and deployment using **BentoML**
* Reproducible machine learning pipeline
* End-to-end project workflow from training to deployment

## Installation

Clone the repository:

```
git clone https://github.com/yourusername/MLops-mlflow-project.git
```

Move to the project directory:

```
cd MLops-mlflow-project
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## Running the Project

Run the training pipeline:

```
python train.py
```

Start MLflow tracking server:

```
mlflow ui
```

Run the application:

```
python app.py
```
